# TryParse

## Получение числа от пользователя с проверкой

```csharp
Console.WriteLine("Введите ваш возраст:"); 
string userInput = Console.ReadLine(); // Пользователь вводит что-то
int age; // Создаём "коробку", куда TryParse положит результат 

// Пытаемся конвертировать. isSuccess будет true или false. 
bool isSuccess = int.TryParse(userInput, out age);

// Теперь проверяем, удалась ли операция
if (isSuccess)
{ // Этот код выполнится, только если пользователь ввёл корректное число
Console.WriteLine($"Через 10 лет вам будет {age + 10} лет.");
}
else
{ // А этот — если ввёл что-то другое Console.WriteLine("Ошибка: вы ввели не число. Пожалуйста, будьте внимательнее.");
}
```

## Получение числа от пользователя с проверкой циклом

```csharp
int age; 
Console.WriteLine("Введите ваш возраст (только цифрами):"); // Будем повторять цикл, ПОКА (! - НЕ) конвертация НЕ удалась

while (!int.TryParse(Console.ReadLine(), out age))
{ // Если мы оказались внутри цикла, значит, TryParse вернул false
Console.WriteLine("Неверный ввод. Пожалуйста, введите возраст цифрами:");
} // Цикл закончится только тогда, когда TryParse вернёт true
// К этому моменту в переменной 'age' уже будет лежать корректное число
Console.WriteLine($"Отлично! Ваш возраст: {age}");
```
