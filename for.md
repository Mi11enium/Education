# Определение
The 'for' loop in Python used for iterating over a sequence or an iterable object.
# Функция
используется для итерации по последовательностям
# Объект воздействия
[[list]], [[tuple]], [[string]], [[set]], [[dictionary]], or any other [[object]]

# Как
Синтаксис
```python
for item in iterable:
```
