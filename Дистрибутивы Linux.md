### **Что Такое Дистрибутив Linux:**

Дистрибутив Linux это не просто "версия" Linux — это **индивидуально собранный набор**:
- Ядро Linux (сердце системы)
- Пакетный менеджер (apt, yum, pacman)
- Предустановленные программы
- Система инициализации (systemd, OpenRC)

### **Популярные дистрибутивы Linux**

### **1. Для новичков**
- **Ubuntu** – один из самых популярных дистрибутивов, основан на Debian, простая установка и удобный интерфейс (GNOME).
- **Linux Mint** – основан на Ubuntu, ещё более дружелюбен к пользователям Windows (Cinnamon/MATE/Xfce).
- **Zorin OS** – похож на Windows/macOS, подходит для перехода с других ОС.
### **2. Универсальные / для разработчиков**
- **Debian** – стабильный и надежный, основа многих других дистрибутивов.
- **Fedora** – использует новейшие технологии (поддерживается Red Hat), часто выбирают разработчики.
- **openSUSE** – мощный дистрибутив с инструментами для администрирования (Tumbleweed – rolling-release, Leap – стабильный).
### **3. Для продвинутых пользователей**
- **Arch Linux** – минималистичный, гибкий, rolling-release (обновления без переустановки).
- **Gentoo** – для полного контроля над системой (сборка из исходников).
- **Slackware** – один из старейших, минимализм и простота конфигурации.
    
### **4. Легковесные дистрибутивы**
- **Lubuntu** – Ubuntu с LXQt, подходит для слабых ПК.
- **Puppy Linux** – очень легкий, может работать прямо с флешки.
- **AntiX** – для очень старых компьютеров (основан на Debian).
    
### **5. Специализированные**
- **Kali Linux** – для пентеста и кибербезопасности.
- **Tails**  – работает через Tor.
- **Parrot OS** – Анонимность и пентест
- **Ubuntu Studio** – для мультимедийного производства (аудио, видео, графика).
