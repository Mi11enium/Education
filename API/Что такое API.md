# Что такое API?
**API (Application Programming Interface)** — это способ, с помощью которого одно приложение может общаться с другим.

API обычно используется для получения или отправки данных

# Методы API

- `GET` — получить данные
- `POST` — отправить/создать
- `PUT` — обновить
- `DELETE` — удалить
