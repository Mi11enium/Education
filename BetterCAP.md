Bettercap написан на языке Go и работает на Linux, macOS, Windows (частично) и даже на Android (с помощью Termux).
Bettercap может делать очень многое. Вот его главные "таланты":

1. **Атака "Человек посередине" (MITM):** Это его основная функция. Bettercap может перехватывать трафик между двумя узлами в сети (например, между жертвой и роутером). Чаще всего для этого используются:
    - **ARP-спуфинг:** для проводных сетей и Wi-Fi (в той же подсети).
    - **Спуфинг DHCP:** для перенаправления трафика.
    - **Спуфинг DNS:** для подмены адресов запрашиваемых сайтов.
2. **Мониторинг сети:**
    - **Сниффинг трафика:** Перехват и анализ сетевых пакетов в реальном времени. Может показывать посещаемые URL, данные форм (HTTP), cookies и т.д.
    - **Анализ хостов:** Постоянно сканирует сеть и показывает список активных устройств, их IP- и MAC-адреса, открытые порты и производителя сетевой карты.
3. **Атаки на Wi-Fi:**
    - Сканирование Wi-Fi сетей и клиентов.
    - Деаутентификация клиентов от точек доступа (отключение устройств от Wi-Fi).
    - Создание поддельных точек доступа (Evil Twin).
4. **Атаки на Bluetooth:** Сканирование и атаки на Bluetooth-устройства.
5. **Модули для атак на более высоких уровнях:**
    - **HSTS Bypass:** Обход защиты HSTS для перехвата HTTPS-трафика.
    - **Прокси-модули:** Могут динамически изменять перехваченный трафик (например, внедрять свой JavaScript в страницы).
    - **Спуфинг SSL-сертификатов.**
6. **Парсинг и анализ данных:** Может извлекать учетные данные из перехваченного трафика (например, при отправке форм без HTTPS).

По умолчанию Bettercap не входит в Kali linux, но его можно легко установить:

Сначала обновите пакеты**:** sudo apt --fix-broken install  
Затем установите сам Bettercap: sudo apt install bettercap

# Первый запуск
sudo bettercap
net.probe on - просмотр всех устройств в сети
net.show - список всех устройств их MAC и IP-адресами 

## Спуфф-атака
set arp.spoof.targets <ip адрес цели> - установление цели для спуффинга
arp.spoof on/off - установка в качестве MITM
net.sniff on/off - прослушивание траффика
