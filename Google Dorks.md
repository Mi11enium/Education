**Google Dorks** (также известные как _Google Hacking_) — это специальные поисковые запросы, которые используют расширенные операторы поисковой системы Google для нахождения скрытой или уязвимой информации в интернете. Эти запросы позволяют фильтровать результаты, чтобы находить конкретные типы данных, которые не отображаются при обычном поиске.

**Google** индексирует огромное количество веб-страниц, включая те, где содержится служебная информация, открытые базы данных, конфигурационные файлы и даже уязвимые системы. С помощью специальных команд (операторов) в поисковой строке можно сузить результаты до определенных параметров. Мы разберем это в практике настолько подробно, насколько только возможно. Важно помнить, что хотя сам по себе этот метод не является взломом, он может использоваться для сбора информации перед атакой. Многие компании случайно оставляют конфиденциальные данные в открытом доступе, и Google индексирует их. Таким образом, Google Dorks — это мощный инструмент для поиска информации, который может использоваться как в легальных целях, так и для кибератак.

- **Базовые операторы:**
    - `site:` – поиск по конкретному сайту.
    - `filetype:` – поиск файлов определённого типа (PDF, DOCX, XLS и т. д.).
    - `intitle:` / `inurl:` – поиск по заголовку или URL.
    - `intext:` – поиск по тексту на странице.
