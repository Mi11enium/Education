**theHarvester** — это инструмент для сбора информации о цели через открытые источники (OSINT). Он ищет email-адреса, поддомены, имена пользователей и связанные с доменом данные. Работает через поисковики вроде Google, Bing, соцсети типа LinkedIn, GitHub, Pastebin и другие публичные базы. Например, можно найти все почтовые адреса компании или поддомены её сайта. Инструмент полезен при подготовке к пентесту или разведке перед атакой — чем больше данных собрано, тем проще найти уязвимости. Запускается из командной строки Kali Linux, поддерживает API Shodan для поиска устройств. Главное — использовать аккуратно, чтобы цель не заметила активного сканирования. Можно комбинировать с другими инструментами вроде Nmap для дальнейшей эксплуатации найденных данных.

Первый ввод (help)
theHarvester -h - выводит основные опции
