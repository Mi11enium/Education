# Упражнения с одномерными массивами
## Вывод элементов с четными индексами

```csharp
	int[] arr = {1, 2, 3, 4, 5};
	
	for (int i = 0; i < arr.Length; i+=2)
	{
		Console.WriteLine(arr[i]);
	}
```

## Вывод элементов с нечетными индексами

```csharp
	int[] arr = {1, 2, 3, 4, 5};
	
	for (int i = 1; i < arr.Length; i+=2)
	{
		Console.WriteLine(arr[i]);
	}
