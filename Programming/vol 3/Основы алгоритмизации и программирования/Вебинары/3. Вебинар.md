# Условный оператор IF
В python отсутствуют скобочки. Вся вложенность определяется отступами.

С одним условием:
~~~python
x = float(input())  
y = float(input())  
  
maximum = x  
if maximum < y:  
    maximum = y  
print(maximum)
~~~

С двумя условиями:
~~~python
x = float(input())  
y = float(input())  
if x < y:  
    maximum = y  
else:  
    maximum = x  
print(maximum)
~~~

С помощью тернарного оператора:
~~~python
x = float(input())  
y = float(input())  
maximum = x if x > y else y  
print(maximum)
~~~
