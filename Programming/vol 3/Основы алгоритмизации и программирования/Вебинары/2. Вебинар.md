# Арифметические операторы

| Operator | Name           | Example |Return|
| -------- | -------------- | ------- | ---- |
| +        | Addition       | x + y   | int  |
| -        | Subtraction    | x - y   | int  |
| *        | Multiplication | x * y   | int  |
| /        | Division       | x / y   | float|
| %        | Modulus        | x % y   | int  |
| **       | Exponentiation | x ** y  | int/float |
| //       | Floor division | x // y  | int |
